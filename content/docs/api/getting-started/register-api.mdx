We're in the process of discontinuing our Register API in favour of a single easy API version. As Personal Access Tokens are now also accepted on our OAuth routes, there is no longer any need for a separate Register API. The current Register API endpoints will still be supported for a while, but we do advise you to migrate to the new API version.

If you have any questions, don't hesistate to contact us at [dev@leat.com](mailto:dev@leat.com).
