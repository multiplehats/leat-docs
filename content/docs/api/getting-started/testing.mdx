As testing is done on the Production environment, take care to not use any actual customer email addresses. If any Automations or Campaigns are activated in your account, the emails will be sent.

Testing your integration can be done simply using our Production environment. We advise you create a new Account - as if you were a new Client - so you can test with a clean dashboard without muddling up actual data with test data.
