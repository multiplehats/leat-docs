All settings live under the **Leat** menu in your WordPress admin. This page documents every available option.

## Settings pages

**General Settings**
- Enable or disable the plugin
- API key (Personal Access Token)
- Shop selection
- Delete settings on deactivation

**Loyalty Program**
- Earn rules configuration
- Rewards and spend rules
- Auto-enroll customers on registration
- Include guests in the loyalty program
- Order statuses that trigger earning
- Order statuses that trigger credit reversal
- Order parts that count toward earning (subtotal, tax, shipping, fees)
- Credits name and balance display format
- Marketing consent subscription
- Only reward known contacts
- Loyalty opt-in checkbox settings

**Dashboard Settings**
- Dashboard page selection
- Logged-in and logged-out titles
- Join program CTA text and title
- Login button text
- Navigation labels (Coupons, Tiers, Earn, Rewards, Activity, Referrals)
- Empty state messages
- Loading state messages
- Show/hide tiers section
- Show/hide referrals section
- Show/hide join program CTA
- My Account tab title

**Referral Settings**
- Referral description text
- Referral link title
- Share button text
- Share text and share title
- People referred label
- Successful referrals label
- Copied notification text

**Gift Card Settings**
- Order status that creates gift cards
- Cancellation statuses
- Accept gift cards at checkout
- Disable recipient email requirement
- Disable recipient message requirement
- All gift card UI text (form title, input placeholder, apply button, balance display, all error messages)

**Voucher Settings**
- Order status that creates vouchers
- Cancellation statuses
- Include voucher code in order emails

**Prepaid Settings**
- Order status that triggers balance top-up
- Reversal statuses
