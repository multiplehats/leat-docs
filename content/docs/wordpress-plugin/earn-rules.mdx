Earn rules control when customers collect credits and how many they earn. You manage them under **Leat > Loyalty Program** in your WordPress admin.

## Topics to be covered

- Types of earn rules (order-based, social actions)
- Setting a fixed credit amount per order
- Setting a percentage of order total
- Configuring which order statuses trigger earning
- Configuring which order parts count (subtotal, tax, shipping, fees)
- Tier-based earning multipliers
- Enabling and disabling individual rules
- Guest order earning
- Auto-enrollment on registration
