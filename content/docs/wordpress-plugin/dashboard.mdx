The customer dashboard is a WordPress page you build and control. The plugin provides a full set of Gutenberg blocks and ready-made patterns so you can put together a loyalty experience without writing any code.

## Available blocks

**Content blocks**
- Loyalty Heading and Loyalty Paragraph for custom copy
- Points Display to show a customer's current credit balance
- Product Points to show how many credits a product earns

**Loyalty blocks**
- Join Program Button for customers who have not yet enrolled
- Earn Rules List showing ways to earn credits
- Spend Rules List showing available rewards to claim
- Coupons List showing a customer's active coupons
- Tiers List showing the tier structure and current progress
- Referral Rewards showing the referral link and share interface

## Available patterns

Drop a full section onto any page from the Gutenberg block inserter under the **Leat** pattern category:

- Complete Loyalty Dashboard: everything on one page
- Loyalty Hero with Image
- Earn Rules Section
- Spend Rules Section
- Coupons Section
- Tiers Section
- Referral Section
